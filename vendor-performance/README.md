# Vendor Performance  

## Northstar Health Operations

---

# Overview

The Vendor Performance subsystem tracks supplier reliability, fulfillment stability, SLA compliance, corrective intervention activity, and operational recovery visibility within Northstar Health Operations.

This subsystem focuses on how vendor behavior affects:

- inventory continuity
- replenishment stability
- operational escalations
- shortage exposure
- operational recovery
- leadership visibility

This subsystem functions as:

# the supplier operational intelligence and recovery governance layer

within the Northstar ecosystem.

# Key Relationships

The subsystem connects to other Northstar domains through shared identifiers.

| Identifier | Relationship |
|---|---|
| vendor_id | Connects vendor records across vendor, inventory, shipment, and escalation data |
| shipment_id | Connects shipment activity to fulfillment and SLA records |
| fulfillment_event_id | Connects fulfillment events to SLA tracking and corrective intervention activity |
| sla_event_id | Connects SLA instability to remediation activity |
| corrective_action_id | Connects remediation lifecycle records |
| related_ticket_id | Connects vendor issues to operational escalation workflows |
| item_id | Connects vendor activity to inventory items |
| location_id | Connects vendor activity to operational locations |

These relationships support:

- cross-system operational visibility
- SQL relationship modeling
- Power BI integration
- enterprise KPI aggregation
